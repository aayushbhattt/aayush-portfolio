VST OpenSource Admin

Centralized dashboard for tracking Vedam's open-source health

React Firebase GitHub API TailwindCSS
GitHub Repo ↗

Overview

An administrative dashboard purpose-built for the Vedam School of Technology organization to manage, track, and encourage open-source contributions. It provides maintainers with a centralized hub to review open-source health metrics, track pull requests, manage issues, and automate CI/CD workflows.

Key Features

  • GitHub API Deep Integration: Pulls live repository statistics (stars, forks, open issues, PR statuses) directly via authenticated GitHub API calls.
  • Contributor Leaderboard: Automatically ranks organization contributors based on merged pull requests and issue resolutions.
  • Role-Based Access Control (RBAC): Integrated with Firebase Auth to ensure only designated VST administrators can approve actions or merge PRs from the dashboard.
  • Automated Issue Tracking: Visualizes issue lifecycles from open to in-progress to closed.
