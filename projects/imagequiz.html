View Live ↗ GitHub Repo ↗

Overview

The Image Quiz Application was developed as a real-time event platform designed to handle simultaneous participant engagement during live sessions.

Successfully deployed in a live environment, it seamlessly handled 10+ concurrent participants competing against a dynamic timer and live scoreboard.

Key Features

  • Real-time Multiplayer: Firebase ensures instantaneous score updates across all clients.
  • Live Leaderboard: Dynamic scoreboard rendering with rank logic sorting users by points.
  • Admin Dashboard: A protected dashboard allowing administrators to control quiz flow, kick participants, and manage questions without writing code.
  • Timer Synchronization: Prevents lag desyncs across clients during timed picture reveals.

Technical Architecture

Frontend Vanilla JS + CSS Custom Properties
Backend & Sync Firebase Realtime Database
