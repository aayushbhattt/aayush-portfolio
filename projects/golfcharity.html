Golf Charity Platform

Comprehensive event management and sponsorship portal for the Golf Charity initiative.

React Firebase Resend API TailwindCSS
GitHub Repo ↗

Overview

A full-stack, end-to-end event management platform engineered specifically for the Golf Charity Initiative. This platform modernizes their entire operation by replacing manual spreadsheet tracking with a unified portal for participant registration, tiered sponsorship handling, and automated communications.

Core Features

  • Participant Dashboard: Seamless registration flows for individual players and foursomes, complete with real-time roster updates.
  • Tiered Sponsorship Management: Dynamic checkout pipelines handling different tiers (Platinum, Gold, Silver, Hole Sponsor) with varying perks and limits.
  • Automated Email Notifications: Integrated with the Resend API to automatically dispatch personalized confirmation receipts, event updates, and tax-deductible invoices to sponsors and players upon successful registration.
  • Admin Command Center: Secure backend panel for event organizers to export player manifests, track incoming revenue against goals, and manage waitlists.

Technical Architecture

The platform relies on a secure React architecture communicating securely via cloud functions to process transactions and trigger emails without exposing API keys.
