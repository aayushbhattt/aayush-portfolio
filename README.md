# Aayush Bhatt — Portfolio

A high-performance, single-page portfolio built for technical professional impact. This site showcases the skills, projects, and achievements of **Aayush Bhatt**, a Full Stack Developer and AI/ML enthusiast.

## 🚀 Features

- **Single Page Architecture**: Seamless navigation without page reloads.
- **Premium Aesthetics**: Dark-themed, tech-focused design with custom typography (Nunito & Cascadia Code).
- **Project Showcases**: Interactive case studies with modal detail views.
- **SEO & Social Optimized**: Fully integrated Meta tags, Open Graph, Twitter Cards, and JSON-LD structured data.
- **Micro-Animations**: Smooth scroll reveal effects and typing animations.
- **Performance First**: Built with Vanilla HTML, CSS, and JS — no heavy frameworks or libraries.
- **Accessibility**: ARIA labels and optimized alt text for screen readers.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla ES6+)
- **Analytics**: Google Analytics (Gtag)
- **Deployment**: Optimized for Hostinger / Netlify / GitHub Pages

## 📂 Project Structure

```text
├── assets/         # Images, certificates, and brand logos
├── js/             # Main application logic and animation scripts
├── styles/         # Modular CSS files (Hero, Navbar, Section-specific)
├── index.html      # Main application shell
├── sitemap.xml     # SEO discovery file
└── robots.xml      # Crawler instructions
```

## 🚢 Deployment (Hostinger)

1. Connect to your Hostinger account via **File Manager** or **FTP**.
2. Upload all files from the root directory (excluding `.gitignore`).
3. Ensure your `favicon.ico` and `preview.png` are in the root directory for proper social sharing.
4. Replace the **Google Analytics ID** in `index.html` with your own tracking code.
